Trabalhando com ID e Classe


Seletores ID

É usado um seletor de ID e um elemento HTML para deixar ele unico e especificamente

Deve ser usado uma única vez na pagina

Exemplo

USANDO UNID

Seletores de classe

O seletor de classe seleciona um elemento com um atributo de classe específico

Pode ser usado mais de uma vez na página

Exemplo:

USANDO UMA CLASSE

USANDO UMA CLASSE

USANDO UMA CLASSE

OUTRO TEXTO
